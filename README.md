# Todo APP

This is new Flutter and Firebase Project in which we will use purely Bloc State management.

## Methodologies and Steps:

--> CI/CD using Github Actions.
--> Firebase Sign UP.
--> Firebase Login.
--> Firebase SignOut.
--> Firebase App Check.
--> Firebase Check User Loggedin Status.
